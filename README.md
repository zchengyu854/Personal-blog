# My Portfolio

## 技术方案

| 技术 | 选型 |
|------|------|
| 框架 | Next.js 14 (App Router) |
| 语言 | TypeScript |
| 样式 | Tailwind CSS |
| 动画 | Framer Motion |
| 图标 | Lucide React |
| 主题 | next-themes (深色/浅色切换) |

## 启动命令

```bash
# 安装依赖
npm install

# 开发模式 (端口 3001)
npm run dev

# 生产构建
npm run build

# 生产启动
npm run start

# 代码检查
npm run lint
```
