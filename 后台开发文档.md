# 个人网站后台开发文档

## 项目概述

本后台系统采用 Python FastAPI 框架开发，用于管理个人网站前端页面的所有内容数据。提供 RESTful API 接口，支持内容的增删改查操作。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Python | 3.10+ | 后端开发语言 |
| FastAPI | 0.100+ | Web框架 |
| Pydantic | 2.0+ | 数据验证 |
| SQLAlchemy | 2.0+ | ORM框架（数据库模块） |
| Alembic | 1.12+ | 数据库迁移工具 |
| Uvicorn | 0.23+ | ASGI服务器 |
| PostgreSQL | 15+ | 主数据库（推荐） |

## 项目结构

backend/
├── app/
│   ├── main.py                 # 应用入口

## 四、统一响应格式

class ApiResponse(BaseModel):
    code: int = 200
    message: str = "success"
    data: Optional[Any] = None

def success_response(data: Any, message: str = "success"):
    return ApiResponse(code=200, message=message, data=data)

def error_response(code: int, message: str):
    return ApiResponse(code=code, message=message, data=None)

## 五、FastAPI应用入口示例

from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(
    title="个人网站后台API",

    allow_methods=["*"],
